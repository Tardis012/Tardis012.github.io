Difference between Frequentist probability and Bayesian probability

Frequentist probability and Bayesian probability represents two different interpretation ways of probability . In my previous blog post(Study Note on the basic knowledge of Probability Theory), the difference between Frequentist probability and Bayesian probability has been explained.

Difference between parametric statistics and nonparametric statistics

Parametric statistics is a branch of statistics which assumes that sample data comes from a population that can be adequately modelled by a probability distribution that has a fixed set of parameters

Nonparametric statistics is based on either being distribution-free or having a specified distribution but with the distribution’s parameters unspecified.

Both of them belong to the content of statistical inference.

Statistic inference is the process of using data analysis to deduce properties of an underlying probability distribution.

Introduction of the”stats” module in Python

The “stats” module is a submodule of the SciPy package in Python. Detailed introduction of SciPy package was listed in this blog post (Algorithm package or framework related to statistics and machine learning-1).

There are two general distribution classes that have been implemented for encapsulating continuous random variables and discrete random variables. A summary of selected methods for discrete and continuous random variables is given in the following table.

Methods Description
pdf/pmf Probability distribution function (continuous) or probability mass function (discrete).
cdf Cumulative distribution function
sf Survival Function (1-CDF)
ppf Percent Point Function (Inverse of CDF)
moment Noncentral moments of n th order
stats Statistics of the distribution (typically the mean and variance, sometimes additional statistics)
fit Fit distribution to data using a numerical maximum likelihood optimization
expect Expectation value of a function with respect to the distribution
interval The endpoints of the interval that contains a given percentage of the distribution(confidence interval)
rvs Random variable samples. Takes as argument the size of the resulting array of samples
mean, median, std, var Descriptive statistics: mean, median, standard deviation, and the variance of the distribution.

Parameter calculation method of probability distribution in the “stats” module

Given the sampled data, we can fit a probability distribution function using maximum likelihood estimation (MLE) with respect to the distribution parameters.

Derive probability distribution from perspective of Bayesian probability and parametric statistics

Parameter calculation method of posterior probability distribution

Parameters of the posterior probability distribution can be estimated based on Maximum A Posteriori (MAP).

Note: From the point of view of Bayesian inference, MLE is a special case of maximum a posteriori estimation (MAP) that assumes a uniform prior distribution of the parameters.

Although Bayes’ theorem describes how to compute the posterior distribution from the prior distribution, for most realistic problems, the calculations involve evaluating high-dimensional integrals that can be prohibitively difficult to compute, both analytically and numerically. This has until recently hindered Bayesian statistics from being widely used in practice. However, with the advent of computational statistics, and the development of efficient simulation methods that allows us to sample directly from the posterior distributions (rather than directly compute it), Bayesian methods are becoming increasingly popular.
