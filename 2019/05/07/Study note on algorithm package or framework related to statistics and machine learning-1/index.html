前言

本篇为Statistics and Machine learning相关的算法包或框架的个人总结笔记 (1/8)。

Topic: the SciPy package.

Four basic concepts

  • Module-模块：比如Numpy。
  • Package-包：比如Pandas、SciPy、Scikit-learn、PyMC、Pylearn2。
  • Library-库。
  • Framework-框架 ：比如TensorFlow、Keras、MXNet、PyTorch、Caffe。

模块层级最小，框架层级最大。

Remark:

Package和Library查资料的时候发现经常混用，除引用时参考原文表述之外，本文其余地方一律不采用Library的说法。

More content can refer to this two article: “Modules vs Packages vs Libraries in Python” and “Python入门之面向对象module，library，package之间区别” and “机器学习库/包的比较” .

Overview of SciPy package

SciPy is an extensive library for applied mathematics and scientific computation based on Python. The following is the complete list of all the modules available in the library (Martins L F, 2014).

Modules in the SciPy package

Module Functionality Functionality(ch)
cluster Clustering algorithms 聚类算法
constants Physical and mathematical constants 物理和数学常量
fftpack Fast Fourier Transform 傅立叶变换
integrate Integration and ordinary differential equations 积分和常微分方程
interpolate Interpolation and splines 插值
io Input and output 数据输入输出
linalg Linear algebra 线性代数程序
ndimage Image processing 多维图像处理
odr Orthogonal distance regression 正交距离回归
optimize Optimization and root-finding 优化
signal Signal processing 信号处理
sparse Sparse matrices 稀疏矩阵
spatial Spatial data structures 空间数据结构和算法
special Special functions 一些特殊的数学函数
stats Statistical distributions 统计
weave C/C++ integration
