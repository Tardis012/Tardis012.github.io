前言

本篇为Statistics and Machine learning相关的算法包或框架的个人总结笔记 (3/8)。

Topic: the TensorFlow framework.

Overview of TensorFlow framework

TensorFlow 是一个使用数据流图(Dataflow Graph) 表达数值计算的开源软件库，支持Python, C++, Java, Go 等多种程序设计语言。

TensorFlow 最初由Google Brain 的研究员和工程师们开发出来，用于开展机器学习和深度神经网络的研究，包括语言识别、计算机视觉、自然语言理解、机器人、信息检索。TensorFlow 系统架构的通用性和灵活性，使其广泛地用于其他科学领域的数值计算。

Remark

  • TensorFlow和PyTorch是最火的两个深度学习框架。
  • TensorFlow 2.0对1.x版本做了一次大的瘦身，使用Keras作为默认高级API，降低了TensorFlow的使用难度。

Modules in the TensorFlow framework

  1. audio module: Public API for tf.audio namespace.
  2. autograph module: Conversion of plain Python into TensorFlow graph code.
  3. bitwise module: Operations for manipulating the binary representations of integers.
  4. compat module: Functions for Python 2 vs. 3 compatibility.
  5. config module: Public API for tf.config namespace.
  6. data module: tf.data.Dataset API for input pipelines.
  7. debugging module: Public API for tf.debugging namespace.
  8. distribute module: Library for running a computation across multiple devices.
  9. dtypes module: Public API for tf.dtypes namespace.
  10. errors module: Exception types for TensorFlow errors.
  11. estimator module: Estimator: High level tools for working with models.
  12. experimental module: Public API for tf.experimental namespace.
  13. feature_column module: Public API for tf.feature_column namespace.
  14. graph_util module: Helpers to manipulate a tensor graph in python.
  15. image module: Image processing and decoding ops.
  16. initializers module: Keras initializer serialization / deserialization.
  17. io module: Public API for tf.io namespace.
  18. keras module: Implementation of the Keras API meant to be a high-level API for TensorFlow.
  19. linalg module: Operations for linear algebra.
  20. lite module: Public API for tf.lite namespace.
  21. lookup module: Public API for tf.lookup namespace.
  22. losses module: Built-in loss functions.
  23. math module: Math Operations.
  24. metrics module: Built-in metrics.
  25. nest module: Public API for tf.nest namespace.
  26. nn module: Wrappers for primitive Neural Net (NN) Operations.
  27. optimizers module: Built-in optimizer classes.
  28. quantization module: Public API for tf.quantization namespace.
  29. queue module: Public API for tf.queue namespace.
  30. ragged module: Ragged Tensors.
  31. random module: Public API for tf.random namespace.
  32. raw_ops module: Public API for tf.raw_ops namespace.
  33. saved_model module: Public API for tf.saved_model namespace.
  34. sets module: Tensorflow set operations.
  35. signal module: Signal processing operations.
  36. sparse module: Sparse Tensor Representation.
  37. strings module: Operations for working with string Tensors.
  38. summary module: TensorFlow component package for providing tf.summary from TensorBoard.
  39. sysconfig module: System configuration library.
  40. test module: Testing.
  41. tpu module: Ops related to Tensor Processing Units.
  42. train module: Support for training models.
  43. version module: Public API for tf.version namespace.
