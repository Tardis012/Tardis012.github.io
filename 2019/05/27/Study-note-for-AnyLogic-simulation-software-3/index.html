前言

本篇为AnyLogic仿真软件的学习总结笔记(3/6)。

具体为：工具栏中的Space markup palette学习。

工具栏学习-Space markup palette

The Space markup palette overview

Space markup palette：包含用于在模型中标记空间的元素。

General space markup elements

  • 路径Path元素定义了Agent的移动路径。

You can select a path as Agent location in the following Process Modeling Library objects’s properties: Delay, Queue, Match, Combine, Seize, Service, Conveyor, Batch, RackStore.

  • 节点Node元素定义了Agent可以驻留的位置。包括3种：Rectangular node、Point node、Polygonal node。
  • Attractor（只能放在Node内部）。
  • Pallet Rack：图形化定义了用于仓库和存储区的托盘架。

GIS elements

  • GIS map：添加GIS地图。
  • GIS point：添加点。每个GIS point都具有以度为单位的纬度和经度坐标。
  • GIS route：添加GIS路线。路线的每个点都有以度为单位的纬度和经度坐标。
  • GIS Region：使用GIS Region元素在地图上标记一些封闭区域。该区域的每个点都具有以度为单位的纬度和经度坐标。
  • Route Provider element enables to define an alternative road network in GIS model plus to the default one defined by the GIS map.

Material Handling

可以定义跟物料运输相关的一些图形化空间标记形状。

Pedestrian

可以定义建筑和行人流方向相关的一些图形化空间标记形状。

Rail

可以定义铁路相关的一些图形化空间标记形状。

Fluid

可以定义液体存储和流动管道等相关的一些图形化空间标记形状。

Road

可以定义道路（比如道路、交叉路口、停车场等）相关的一些图形化空间标记形状。
