R new package recommended from Jan to Mar 2019

Posted on 2019-04-28 | In Software

前言

本篇记录2019年1-3月R新包中自己觉得有用的。

R New Package Recommended in Jan 2019

1月份，共有153个R新包收录于CRAN。

Hydrometeorology

  1. ecmwfr：针对欧洲中期天气预报中心(ECMWF)的公共和限制数据集和哥白尼的气候数据(CDS)，提供程序化操作界面，同时允许用户下载天气预报和气候数据。
  2. nhdR： 提供使用国家水文数据集(NHD)的工具，包括查询、下载和联网NHD与NHDPlus数据集。

Statistics and Machine learning

  1. concurve： 提供函数来计算各种统计测试的置信区间(相容性/协调性)，以及它们对应的P值和S值。
  2. FiRE： 提供在海量数据集中查找异常值和稀有实体的算法。
  3. mimi： 提供函数，用以估计具有缺失值的混合数据集中的主要影响。
  4. zoomgrid： 实现了一种带缩放的网格搜索算法，以帮助解决目标函数域内存在多个局部最优的困难优化问题。
  5. apcf： 实现了自适应对相关函数，将对相关函数的概念从点模式转移到有限尺寸和不规则形状的对象。
  6. foto： 实现傅立叶变换纹理排序方法，该方法使用径向平均二维傅立叶谱的主成分分析来表征图像纹理。

Others

  1. tor： 提供允许用户同时导入多个文件的功能。
  2. vitae： 提供多个模板和功能以简化简历的制作和维护。

R New Package Recommended in Feb 2019

2月份，共有151个R新包收录于CRAN。

Hydrometeorology

  1. noaaoceans： 提供访问国家海洋和大气管理局(NOAA) API的工具。

Statistics and Machine learning

  1. geosample： 提供构建抽样设计的功能。
  2. missCompare： 提供方法用于测试和比较模拟、实际数据上的各种缺失数据插补算法。
  3. OutlierDetection： 提供多种方法来检测异常值，包括基于模型的方法、基于距离的方法、基于密度的方法等。
  4. fastNaiveBayes： 提供朴素贝叶斯分类器的快速实现，主要基于Schneider(2003)的论文。
  5. Spectrum： 为单视图和多视图数据实现快速、自适应的谱聚类算法。
  6. r.blip： 提供从包含数千个变量的数据集中学习贝叶斯网络的功能，包括父集识别、一般结构优化、有界树宽度结构的算法优化、不完整数据集的结构学习等。
  7. RTML： 实现10种正则化多任务学习算法的有效求解器，适用于回归、分类、联合特征选择、低秩学习、稀疏学习和网络合并等。
  8. Rlgt： 提供使用rstan的函数，用于拟合全局趋势模型进行时间序列预测，这些模型是贝叶斯归纳和某些指数平滑模型的扩展。
  9. tsfeatures： 如Hyndman等人所述，实现从时间序列数据中提取各种特征的多种方法。

Others

  1. qs： 提供快速写入和读取任何R对象到磁盘和从磁盘读取的功能。

R New Package Recommended in Mar 2019

3月份，共有233个R新包收录于CRAN。

Statistics and Machine learning

  1. runstats： 提供快速计算时间序列样本统计数据的方法，包括：平均值、标准差、相关性、协方差等。
  2. DistributionOptimization： 支持借助GA包的遗传算法来拟合高斯混合模型。
  3. mcmcabn： 实现用于有向无环图(DAG)的MCMC采样器。
  4. networkABC： 实现了一种新的多级近似贝叶斯计算(ABC)算法，用于解译网络数据和评估网络参与者之间推断链接的强度。
  5. tacmagic： 提供便于分析正电子发射断层(PET)扫描时间活动曲线(TAC)数据的功能。

Others

  1. predict3d： 通过ggplot2和rgl为多个回归模型提供二维和三维的绘图功能。

Remarks

R is a very powerful, popular environment for statistical computing and graphics, freely available for Mac OS X, Windows, and UNIX systems. Knowing how to use R is an extremely useful skill. R and various supporting information can be obtained at https://www.r-project.org. RStudio is an excellent alternative interface for R, freely available at https://www.rstudio.com.
