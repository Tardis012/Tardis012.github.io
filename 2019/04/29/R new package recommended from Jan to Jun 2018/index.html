R new package recommended from Jan to Jun 2018

Posted on 2019-04-29 | In Statistics and Machine learning

前言

本篇记录2018年1-6月R新包中自己觉得有用的。

R New Package Recommended in Jan 2018

1月份，共有200个R新包收录于CRAN。

Hydrometeorology

  1. washdata：支持获取城市供水和卫生调查数据，该数据由城市贫民清洁水处理联盟机构(WSUP)提供。

Statistics and Machine learning

  1. detpack：提供部分功能，用于对大数据集进行密度估计，同时使用分布元素树生成条件/无条件随机数。
  2. gnorm：提供获得广义正态/指数功率分布概率、分位数、密度和随机偏差的函数。
  3. IROmiss：提供一种通用算法，即插值正则化优化(IRO)算法，用于处理高维缺失数据问题。
  4. stranger：提供一个解决框架，针对无监督异常检测问题。
  5. natural：在高维线性模型中，实现两种误差方差估计方法。
  6. breakDown：实现分解图，显示模型中的每个变量的贡献。
  7. KRIG：提供Kriging模型和空间统计的各种方法的功能，包括使用再生核希尔伯特空间的多元敏感性分析和Sobol指数的计算。
  8. OpVar：使用最大似然估计和贝叶斯方法，提供用于针对操作风险建模的函数。
  9. dalmatian：提供一个有效的功能，借助JAGS自动拟合GLM模型。
  10. dirichletprocess：允许创建Dirichlet过程对象，这些对象可以用作无限混合模型。包括密度估计、泊松过程强度推理、分层建模和聚类等。
  11. CRPClustering：提供一种使用中国餐馆过程Pitman(1995)的聚类方法，该方法不需要事先确定聚类数量，同时提供相关函数计算类的模糊性作为熵值Yngvason(1999)。
  12. multiROC：通过微观和宏观平均计算ROC曲线下面积，提供工具来解决多个类别的分类问题。
  13. reinforcelearn：实现强化学习环境和算法 Sutton＆Barto(1998)。
  14. TSrepr：提供时间序列的表示方法（如：降维、预处理、特征提取等）。
  15. santaR：为短时间序列分析提供了一个图形化的自动化流水线功能，旨在适应异步时间采样、个体间变异性、噪声测量和变量较多等问题。
  16. TSstudio：提供一组交互式可视化工具，支持ts、mts、zoo和xts对象的时间序列分析，包括用于预测模型性能、时间序列交互式图表和季节性图表的可视化函数等。

R New Package Recommended in Feb 2018

2月份，共有171个R新包收录于CRAN。

Hydrometeorology

  1. photobiologySun：提供外星太阳光谱辐照度、地平面太阳光谱辐照度的数据。
  2. CENFA：提供针对空间数据气候和生态因素分析的工具，包括对气候变化引起的物种敏感性、暴露、脆弱性等变化的空间数据可视化。

Statistics and Machine learning

  1. CholWishart：提供了针对几种分布的抽样函数，如威沙特分布（Wishart）随机变量的斯基（Cholesky）因式分解、反威沙特分布（Wishart）和反威沙特分布随机变量的斯基（Cholesky）分解。
  2. GpGp：提供对高斯过程进行预测和条件模拟的功能，并提供对于欧几里德域和球体空间的空间数据、时空数据的协方差函数。
  3. kdensity：提供使用参数初值和不对称内核进行单变量非参数密度估计的方法。
  4. rsimsum：提供汇总模拟结果的功能，并用于计算蒙特卡罗标准误差。
  5. SimCorrMix：提供生成具有特定相关矩阵的连续（正态、非正态或混合分布）、二元、有序变量的方法，或具有混合分布的连续变量的方法。
  6. DALEX：提供各种解释器，帮助理解机器学习模型中输入变量与模型输出之间的关联关系。
  7. tree.bins：允许用户通过从函数：rpart()（rpart包）派生的决策树方法，对因子变量进行重新分类(Hastie et al. 2009)。
  8. kmed：基于Park和Jun(2009)的方法，实现了基于距离的k-medoids聚类算法，并通过重新排序矩阵算法生成热图来验证聚类簇。
  9. lolR：在进行监督学习技术时，对于维度超过样本大小的情况，实现了最佳低秩投影算法，从而获得数据的较低维表示。
  10. projpred：提供了为广义线性模型进行投影预测特征选择的方法(Piironen and Vehtari, 2017)，该包与rstanarm兼容。
  11. rosqp：支持与OSQP求解程序绑定，来解决稀疏凸二次规划问题。
  12. tstools：支持绘制官方统计时间序列，可以便捷的增加图例、高亮窗口、带有正负贡献的堆积条形图及其他选项。
  13. geozoning：提供用于评估分区质量的分区方法和数字标准。
  14. riskyr：提供以概率或频率来表达风险相关信息的功能，使风险读写的教学和培训更加透明。

R New Package Recommended in Mar 2018

3月份，共有171个R新包收录于CRAN。

Hydrometeorology

  1. daymetr：提供Daymet气候数据的编程接口，并支持相关分析。
  2. NOAAWeather：提供从所有NOAA站点检索实时天气数据的功能，并支持绘制时间序列、箱线图、热图和地理空间图等图形。
  3. foreSIGHT：提供工具来创建水文气候场景、压力测试、可视化等系统，用于气候变化影响的评估。

Statistics and Machine learning

  1. iTOP：提供函数来推断不同数据集之间的拓扑结构关系，该方法是基于RV系数的扩展。
  2. iml：提供了部分具备可解释性的方法，用于分析机器学习模型的行为和预测结果。包括特征重要性、局部依赖图、Shapley值和树替代模型等。
  3. fmlogcondens：实现了多元对数凹函数族最大似然估计的快速求解。
  4. spGARCH：提供了部分功能，用于分析空间和时空自回归条件异位、空间ARCH类型过程的模拟、spARCH模型参数的准极大似然估计等问题。
  5. tsfknn：提供使用最近邻回归预测时间序列的功能。

Others

  1. cranly：提供用于清理、组织、总结和可视化CRAN包信息的功能，以及构建包指示网络(依赖、导入、建议、增强)和协作网络的功能。
  2. smovie：使用rpanel包创建交互式动画，帮助学生理解统计概念。

R New Package Recommended in Apr 2018

4月份，共有212个R新包收录于CRAN。

Statistics and Machine learning

  1. diffeqr：支持对微分方程计算工具的调用接口(DifferentialEquations.jl)，提供了针对常微分方程(ODE)、随机微分方程(SDE)、时滞微分方程(DDE)、微分代数方程(DAE)等方程的高性能方法。
  2. cultevo：提供测量信号系统组成的工具：信息论测量(2016)、距离矩阵相关性的Mantel检验(1983)等。
  3. konfound：实现了反映灵敏度分析最新进展的方法，包括计算稳健性指数的函数。
  4. RNGforGPD：提供生成遵循广义泊松分布的单变量和多变量数据的函数。
  5. SmartEDA：提供自动描述数据结构和关系的功能，以便于探索性数据分析。
  6. WeMix：提供运行混合效果模型的功能，其中包括每个级别的权重。遵循Rabe-Hesketh等人所提出的方法(2006)，利用自适应正交的方法进行模型训练。
  7. featurefinder：通过使用rpart分类和回归树对模型残差进行详细分析，实现了一种用于查找模型特征的方法。
  8. iilassso：用可独立解释的Lasso实现用于拟合线性、逻辑回归模型的高效算法。
  9. live：通过了解关键因素，实施解释复杂机器学习模型的方法，详见 Ribeiro et al. (2016) and Staniak and Biecek (2018)。
  10. varrank：提供用于执行变量排序和特征选择的计算工具箱。
  11. anomalize：实现一个“整洁”工作流程，用于检测时间序列数据中的异常情况，支持分解时间序列、检测异常和将多个时间序列的正常数据与异常数据进行分离等功能。
  12. LPWC：提供函数-基于加权相关和引入滞后，进行时间序列的距离度量。
  13. ERSA：提供函数对回归计算的结果进行展示，同时该功能可以集成到shiny中进行可视化显示。

Others

  1. tabr：支持通过R代码利用乐谱软件(LilyPond)创建吉他谱的功能，可根据标记语言生成高品质的乐谱。
  2. promises：提供了在R中进行异步编程的基本抽象。
  3. prepplot：提供一个单独的函数，用于创建基础绘图区域，可以轻松制作自定义图形。

R New Package Recommended in May 2018

5月份，共有215个R新包收录于CRAN。

Statistics and Machine learning

  1. dqrng：提供具有良好统计特性的快速随机数生成器，包括Mersenne-Twister的64位版本、pcg64、Xoroshiro128和Xoroshiro256。
  2. catch：提供了对高维张量（多维数组）进行分类和变量选择的功能。
  3. ODEsensitivity：提供使用ODEnetwork包对常微分方程(ODE)模型进行灵敏度分析的功能，支持模拟二阶ODE网络。
  4. bioacoustics：提供分析录音和自动提取动物声音的功能，包括处理各种格式的音频录音（如WAV, WAC, MP3, ZC）、过滤有噪声的文件、检测和提取自动的声学特性以进行进一步的分析等。
  5. glmmboot：提供两个函数，为众多统计模型提供自助重采样功能。函数BootGlmm()：如果存在随机效应，则执行块重采样；如果不存在，则执行大小写重采样。函数BootCI()：将模型的输出转换为置信区间和p值。

  6. survBootOutliers：在生存环境中提供三种新的基于一致性的异常值检测方法。

  7. vinereg：利用参数或非参数的pair-copulas实现D-vine分位数回归模型，参见Kraus and Czado (2017) and Schallhorn et al. (2017)。

  8. ggdistribute：对ggplot2进行扩展，绘制后验或其他类型的单峰分布。

R New Package Recommended in Jun 2018

6月份，共有144个R新包收录于CRAN。

Statistics and Machine learning

  1. rcrane：提供优化算法来估计模型中的系数，比如线性回归和神经网络，包括批量梯度下降、随机梯度下降、小批量梯度下降等方法。
  2. betaboost：使用与betareg Cribari-Neto和Zeileis(2010)相同的参数化方法，对潜在的高维数据Mayr等(2018)进行增强beta回归。
  3. bfw：提供使用马尔可夫链蒙特卡罗与JAGS采样器进行贝叶斯分析的框架。
  4. GLMMadaptive：利用自适应高斯求积规则，给出了在最大似然条件下拟合单个分组因子的广义线性混合模型的函数。
  5. glmmfields：实现具有鲁棒随机场的广义线性混合模型，应用于时空建模。
  6. vimp：提供函数，用于计算高维和低维非参数变量重要性度量的点估计和有效置信区间。
  7. vsgoftest：支持对包括均匀分布、高斯分布、指数分布等分布族实施Vasicek和Song拟合优度检验（基于Kullbach-Leibler散度）。
  8. AdaSampling：实现了自适应采样过程，支持进行正向无标记学习和类标签噪声学习。
  9. ratematrix：提供使用马尔可夫链蒙特卡罗(MCMC)估算进化速率矩阵的函数，详见Caetano和Harmon(2017)中所述。
  10. anomaly：实现了CAPA (Collective And Point Anomaly)算法，用于检测时间序列中的异常值，该算法详见Fisch, Eckley and Fearnhead (2018)。
  11. cloudml：提供接口，支持在R环境中使用谷歌云机器学习平台。
  12. skynet：基于美国运输统计局(BTS)的公开数据，支持进行航空运输数据统计及其它分析。
  13. pectralAnalysis：提供光谱分析工具包，支持进行预处理、可视化和过程分析。

Others

  1. tiler：提供从地理空间地图文件和简单的非地理地图图像中，创建地理地图的功能。
