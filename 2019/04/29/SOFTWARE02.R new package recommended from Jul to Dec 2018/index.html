SOFTWARE02.R new package recommended from Jul to Dec 2018

2019-04-29

R

前言

本篇记录2018年7-12月R新包中自己觉得有用的。

R New Package Recommended in Jul 2018

7月份，大约有251个R新包收录于CRAN。

Hydrometeorology

  1. weathercan： 提供从加拿大环境和气候变化网站下载历史天气数据的工具，可以在特定日期范围内从多个工作站下载数据，并自动处理为单个数据集。

Statistics and Machine learning

  1. adaptMT： 实现了自适应的p值阈值(AdaPT)，包括一个允许用户指定任何算法来学习本地错误发现率的框架、以及一个实现特定算法的便捷函数池。

  2. MultiFit： 提供测试两个随机向量的独立性的功能。

  3. robmixglm： 使用混合方法实现稳健的广义线性模型（GLM）,如Beath（2018）中所述。

  4. bigdatadist： 提供函数来计算概率、曲线样本的熵、函数数据的距离和深度度量以及高维数据的广义马氏核距离。

  5. L0Learn： 提供了一个优化的工具包，用于近似解决L0规则化的学习问题。

Others

  1. dbx： 提供选择、插入、更新和删除数据操作的功能，目前支持的数据库主要有PostgreSQL,MySQL, SQLite。

  2. metaDigitise： 提供从研究论文中的已发表数据中提取、汇总和数字化数据的功能。

R New Package Recommended in Aug 2018

8月份，共有160个R新包收录于CRAN。

Statistics and Machine learning

  1. bcaboot： 提供函数以一种几乎自动的方式计算bootstrap置信区间。
  2. bivariate ： 提供用于构造、绘制双变量概率分布的便捷函数。
  3. dials： 提供用于创建无法从数据直接估计模型参数的工具。

  4. fieldRS： 利用Olofsson等人提出的最佳实践，为遥感现场工作提供支持。

  5. Rnmr1D ： 提供从自由感应衰变的原始数据处理质子核磁共振谱的功能，参见Jacob et al。 (2017) 。

Others

  1. tidybayes： 提供组合数据的功能，以整齐的数据格式从贝叶斯模型（JAGS、Stan、rstanarm、brms…）中提取、操作和可视化。
