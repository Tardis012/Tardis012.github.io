SOFTWARE02.R new package recommended from Jul to Dec 2018

Veröffentlicht am 2019-04-29 | Bearbeitet am 2019-04-30 | in Software

前言

本篇记录2018年7-12月R新包中自己觉得有用的。

R New Package Recommended in Jul 2018

7月份，大约有251个R新包收录于CRAN。

Hydrometeorology

  1. weathercan： 提供从加拿大环境和气候变化网站下载历史天气数据的工具，可以在特定日期范围内从多个工作站下载数据，并自动处理为单个数据集。

Statistics and Machine learning

  1. adaptMT： 实现了自适应的p值阈值(AdaPT)，包括一个允许用户指定任何算法来学习本地错误发现率的框架、以及一个实现特定算法的便捷函数池。

  2. MultiFit： 提供测试两个随机向量的独立性的功能。

  3. robmixglm： 使用混合方法实现稳健的广义线性模型(GLM)，如Beath(2018)中所述。

  4. bigdatadist： 提供函数来计算概率、曲线样本的熵、函数数据的距离和深度度量以及高维数据的广义马氏核距离。

  5. L0Learn： 提供了一个优化的工具包，用于近似解决L0规则化的学习问题。

Others

  1. dbx： 提供选择、插入、更新和删除数据操作的功能，目前支持的数据库主要有PostgreSQL，MySQL， SQLite。

  2. metaDigitise： 提供从研究论文中的已发表数据中提取、汇总和数字化数据的功能。

R New Package Recommended in Aug 2018

8月份，共有160个R新包收录于CRAN。

Statistics and Machine learning

  1. bcaboot： 提供函数以一种几乎自动的方式计算bootstrap置信区间。
  2. bivariate ： 提供用于构造、绘制双变量概率分布的便捷函数。
  3. dials： 提供用于创建无法从数据直接估计模型参数的工具。

  4. fieldRS： 利用Olofsson等人提出的最佳实践，为遥感现场工作提供支持。

  5. Rnmr1D ： 提供从自由感应衰变的原始数据处理质子核磁共振谱的功能，参见Jacob et al. (2017) 。

Others

  1. tidybayes： 提供组合数据的功能，以整齐的数据格式从贝叶斯模型(JAGS、Stan、rstanarm、brms…)中提取、操作和可视化。

R New Package Recommended in Sep 2018

9月份，共有126个R新包收录于CRAN。

Statistics and Machine learning

  1. ahpsurvey ：实现层次分析过程，一个由Saaty(1987)引入的通用多准则决策工具，它允许决策者权衡属性并评估呈现给属性的备选方案。
  2. survivalAnalysis ：实现高级接口封装以支持生存分析，包括Kaplan-Meier分析、对数秩检验、Cox回归。
  3. salty ：允许用户获取真实或模拟数据，并将其与常见的错误相结合，例如伪OCR错误、Unicode问题、带有荒谬标点符号的数字字段、错误日期等。
  4. ungroup ：提供实现惩罚复合链路模型的功能，有效地估计粗略分箱数据的平滑分布。
  5. empirical ：实现经验单变量概率密度函数（连续函数）和经验累积分布函数（阶梯函数或连续函数）。
  6. rmatalog ：实现了metalog分布，这是一种现代的、高度灵活的、数据驱动的分布。
  7. basisMCMCplots ：提供用于检查来自NIMBLE软件包接口的单链、多链后验MCMC样本的功能。
  8. samplingBigData ：提供对大数据集进行采样的方法，包括在多维空间中进行空间平衡采样。
  9. embed ：提供使用简单的广义线性模型、非线性模型将因子预测变量转换为一个或多个数值表示的函数。
  10. Opt4PL ：提供获取和评估3、4和5种参数逻辑回归模型的各种最佳设计的功能。
  11. splinetree ：提供使用样条投影方法构建回归树、随机森林的功能。
  12. rwavelet ：提供执行小波分析（正交和平移不变变换）的功能，同时进行数据压缩、去噪声。
  13. bayesdfa ：使用Stan实现贝叶斯动态因子分析，这是一种多变量时间序列的降维工具。
  14. tbrf ：提供基于日期和时间窗口的滚动统计函数，而不是n-滞后观察。
  15. UKgrid ：提供自2011年以来英国国家电网需求数据（高压电力传输网络）。

Others

  1. customLayout ：提供了graphics::layout()的扩展版本，允许用户通过组合更简单的布局为多个元素创建复杂的绘图区域。
  2. ggTimeSeries ：提供其他时间序列可视化，如日历热图、Steam Graph流图等。

R New Package Recommended in Oct 2018

10月份，185个R新包收录于CRAN。

Hydrometeorology

  1. nasapower v1.02：实现POWER（全球能源资源预测），NASA全球气象，地面太阳能和气候数据API的接口。
  2. ClimProjDiags v0.0.1：提供用于计算气候分析的度量和指数，比较模型以及将它们组合成集合的功能。关于异常协议，昼夜温度，极端指数以及热浪和冷波持续时间都有一些小插曲。

Statistics and Machine learning

  1. lognorm v0.1.3：估计分布参数并计算对数正态分布的矩和其他基本统计量Limpert et al.(2001)，并且还提供了几个相关的对数正态分布变量Lo(2013)之和的分布的近似值。关于聚合相关随机变量和另一个关于近似和的小插图。
  2. matrixNormal v0.0.0：提供计算Matrix Normal分布的密度，概率和随机偏差的函数。见Iranmanesh et al.(2010)。
  3. ppgmmga v1.0.1：基于高斯混合模型实现降维的投影追踪算法。该小品提供了封装的快速浏览。
  4. SAMpack v0.1.1：实现随机逼近蒙特卡罗(SAMC)采样器，能够从多模或双重难处理分布中采样。参见Liang et al. (2010)对该方法的完整介绍，以及介绍该软件包的插图。
  5. gcForest v0.2.7：提供深度森林的Python实现的API接口，深度学习的替代方法。该算法在Zhou和Feng(2017)中有描述，并有一个简短的包教程。
  6. sboost v0.1.0：提供基于C ++的快速实现Freund和Schapire的Adaptive Boosting(AdaBoost)算法，并包括分类器评估，预测和交叉验证的方法。
  7. RcppEnsmallen v0.1.10.0.1：实现基于C ++的Ensmallen数学优化库的接口，该库提供了一组简单的抽象，用于编写要优化的目标函数。优化器包括完整批次梯度下降技术，小批量技术，无梯度优化和约束优化。
  8. MachineShop v0.2.0：为机器学习模型拟合，预测，性能评估和结果表示提供通用接口。有一个简介和一份关于实施公约的说明。
  9. mlflow v0.8.0：提供一个MLflow开源平台的接口，用于完整的机器学习生命周期，支持安装，跟踪实验，运行项目和保存模型。

R New Package Recommended in Nov 2018

11月份，171个R新包收录于CRAN。

Statistics and Machine learning

  1. Riembase：实现了多种算法来估计基本统计量，包括用于多值数据的Frechet均值和几何中值。
  2. LindleyPowerSeries：提供计算Lindley Power Series分布的概率密度函数、累积分布函数、危险率函数、分位数函数等功能。
  3. mixsqp：提供基于序列二次规划(SQP)的优化算法(Kim et al. 2012)，用于有限混合模型（成分密度已知）中混合比例的最大似然估计。
  4. modi：实现将样本设计考虑在内的算法来检测多元异常值。
  5. polylabelr：对C ++库polylabel进行封装，提供了一个有效的例程来查找多边形不可访问的近似极点。
  6. pivmet：为了解决贝叶斯混合模型中的标签切换问题，提供了一组关键算法来重新标记MCMC链，同时提供函数初始化经典k均值算法的中心，以便获得更好的聚类解决方案。

Others

  1. countcolors：提供计算图像中颜色的相关函数，以及将目标像素更改为不同颜色图像的掩码版本。

R New Package Recommended in Dec 2018

12月份，157个R新包收录于CRAN。

Hydrometeorology

  1. bioRad ：提供从天气雷达数据中提取、可视化和汇总鸟类和昆虫在空中运动行为的功能。
  2. traitdataform ：提供帮助处理生态特征数据，同时支持应用Schneider等人描述的生态特征数据标准术语(2018年)。
  3. solartime ：提供计算太阳位置、日出和日落时间的功能。

Statistics and Machine learning

  1. mistr ：提供混合分布的计算框架，重点关注复合模型。
  2. mulitRDPG ：提供适合多随机点积图模型的函数，并测试两个网络是否来自同一个分布。
  3. ocp ：针对单变量或多变量数据，实现了Adams和Mackay(2007)的贝叶斯在线变点检测方法及高斯和泊松概率模型。
  4. probably ：提供用于后处理类概率估计的工具。
  5. ar.matrix：支持借助精度矩阵、Choleski分解来模拟自回归数据.
  6. MTLR ：实现了Yu等人提出的多任务Logistic回归(MTLR)(2011年)。
  7. pomdp：提供 pomdp-solve的接口， 它是部分可观察马尔可夫决策过程(POMDP)的求解器。
  8. ParBayesianOptimization ：提供了一个框架，用于根据Snoek等人(2012)描述的方法优化贝叶斯超参数。
  9. mfbvar ：提供用于估计明尼苏达或稳态先验的混合频率贝叶斯向量自回归(var)模型的函数，如Schorfheide和Song(2015)或Ankargren、Unosson和Yang(2018)使用的模型。
  10. NTS ：提供模拟、估计、预测和识别非线性时间序列模型的函数。
