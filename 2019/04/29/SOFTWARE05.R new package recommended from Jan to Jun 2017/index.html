SOFTWARE05.R new package recommended from Jan to Mar 2017

Posted on 2019-04-29 | In Software

前言

本篇记录2017年1-3月R新包中自己觉得有用的。

R New Package Recommended in Jan 2017

1月份，共有251个R新包收录于CRAN。

Hydrometeorology

  1. getCRUCLdata v.1.1: Provides functions that automate downloading and importing climatology data from University of East Anglia Climate Research Unit (CRU). There is vignette to get you started.
  2. metScanR v0.0.1: Provides functions for mapping, and gathering meteorological data from various US surface networks: COOP, USCRN, USRCRN, AL-USRCRN, ASOS, AWOS, SNOTEL, SNOTELLITE, SCAN, SNOW, and NEON.

Statistics and Machine learning

  1. MAVE v0.1.7: Implements the MAVE (Minimum Average Variance Estimation) method of dimension reduction. Look here for the math and here for examples.
  2. EnsCat v1.1: Implements various clustering methods for categorical data. See the website for examples and the paper for the details.
  3. mfe v0.1.0: Provides functions to extract meta-features from datasets to support the design of recommendation systems. The vignette provides examples.
  4. tsdecomp v0.2: Implements ARIMA model-based decompositions for quarterly and monthly time series data. The vignette describes the math.

Others

  1. PeriodicTable v0.1.1: Contains a dataset of the properties of chemical elements.

R New Package Recommended in Feb 2017

2月份，共有145个R新包收录于CRAN。

Hydrometeorology

  1. rnaturalearthdata v0.1.0: Contains the world vector map data used in Natural Earth.

Statistics and Machine learning

  1. liquidSVM v1.0.1: Provides several functions to support a fast support vector machine implementation. There is a demo vignette and supplemental installation documentation.
  2. OOBCurve v0.1: Provides a function to calculate the out-of-bag learning curve for random forest models built with the randomForest or ranger packages.
  3. BayesCombo v1.0: Implements Bayesian meta-analysis methods to combine diverse evidence from multiple studies. The vignette provides a detailed example.
  4. BayesianTools v0.1.0: Implements various Metropolis MCMC variants (including adaptive and/or delayed rejection MH), the T-walk, differential evolution MCMCs, DREAM MCMCs, and a sequential Monte Carlo particle filter, along with diagnostic and plot functions. The vignette will get you started.
  5. IMIFA v1.1.0: Provides flexible Gibbs sampler functions for fitting Infinite Mixtures of Infinite Factor Analysers and related models, introduced by Murphy et al. in a 2017 paper. The vignette shows examples.
  6. lmvar v1.0.0: Provides functions to run linear regressions in which both the expected value and variance can vary by observation. Vignettes provide an introduction and explain the details of the math.
  7. fpp2 v2.0: Provides all of the data sets used in the online book Forecasting: Principles and Practice.
  8. sonify v0.1-0: Contains a function to transform univariate data, sampled at regular or irregular intervals, into a continuous sound with time-varying frequency. The function is intended as a substitute for R’s plot function to simplify data analysis for the visually impaired.

R New Package Recommended in Mar 2017

3月份，共有216个R新包收录于CRAN。

Statistics and Machine learning

  1. kerasR v0.4.1: Provides an interface to the Keras Deep Learning Library, which provides specifications for describing dense neural networks, convolution neural networks (CNN), and recurrent neural networks (RNN) running on top of either TensorFlow or Theano. The vignette contains examples.
  2. modeval v0.1.2: Allows users to easily compare multiple classifications models built with caret functions for small data sets. The vignette provides examples.
  3. supc v0.1: Implements the self-updating process clustering algorithms proposed in Shiu and Chen. The vignettecontains examples.
  4. mlrMBO v1.0.0: Provides a toolbox for Bayesian, model-based optimization. There is an introduction and vignettes for Mixed Space Optimization, Noisy Optimization, and Parallelization.
  5. rjmcmc v0.2.2: Provides functions to perform reversible-jump MCMC with the restriction introduced by Barker & Link. The vignette shows how to calculate posterior probabilities from the MCMC output.
  6. MonteCarlo v1.0.0: Simplifies Monte Carlo simulation studies by providing functions that automatically set up loops to run over parameter grids, parallelize the computations, and generate output in LaTeX tables. The vignette shows how to use it.
