前言

本篇记录定下来主攻PyTorch深度学习框架的原因。

深度学习概念

深度学习的概念源于人工神经网络的研究，以深度神经网络为基本模型（深度可以理解为多层），通过组合低层特征形成更加抽象的高层表示，以发现数据的分布式特征表示。深度学习模型的典型例子是前馈深度网络或多层感知机（multilayer perceptron, MLP）。

图1. 神经网络架构展示（http://www.asimovinstitute.org/wp-content/uploads/2019/04/NeuralNetworkZoo20042019.png）

深度学习框架

之前为了了解package和framework的概念区别、目前有哪些好用的机器学习算法package、目前有哪些流行的深度学习框架，写了8篇博文（Study note on algorithm package or framework related to statistics and machine learning系列）。

深度学习框架有哪些

TensorFlow是Google Brain的研究员和工程师们开发出来的。

PyTorch是一个基于Torch的Python开源深度学习框架，主要由Facebook的人工智能研究小组开发。

MXNet是小团体一起合力做出来的平台。

Keras以 TensorFlow, CNTK, 或 Theano 作为底层，上层提供简单接口，主要由François Chollet一人之力做的。

还有CNTK、Theano、Caffe等。

选PyTorch的原因

  1. PyTorch更有利于研究人员、爱好者、小规模项目等快速搞出原型。而TensorFlow更适合大规模部署，特别是需要跨平台和嵌入式部署时。

操作系统-选用Ubuntu

Ubuntu是基于linux内核的桌面PC操作系统，易用性强、使用人数多、教程多、生态好、各个深度学习框架的官方支持好（比如TensorFlow GPU只介绍了在 Ubuntu 下的安装方法）。

为了能够使用大多数深度学习工程师开源的代码，建议使用 Ubuntu 作为运行代码的机器。
