深度学习框架

深度学习框架有哪些

TensorFlow

TensorFlow是由Google Brain的研究员和工程师们开发出来的。

TensorFlow的名字来源于其计算过程中的操作对象为多维数组，即张量（Tensor）。

PyTorch

PyTorch是由Facebook、NVIDIA、Twitter 等公司开发维护的Python深度学习框架，其前身为Lua 语言的Torch。

MXNet

MXNet由亚马逊、华盛顿大学和卡内基梅隆大学等开发维护。

Keras

Keras 以TensorFlow, CNTK, 或 Theano 作为底层，上层提供简单接口，主要由François Chollet一个人做的（已被谷歌收购）。

CNTK

CNTK由微软开发，比较小众。

Caffe

Caffe全称为Convolutional Architecture for Fast Feature Embedding，是一个卷积网络模型的计算框架，所要实现的网络结构可以在配置文件中指定，不需要编码。Caffe 是用C++和Python 实现，由华人开发的，主要用于计算机视觉。

Theano

Theano是蒙特利尔大学开发的Python 工具包，目前已停止维护。

Chainer

Chainer是一个最早采用动态计算图的神经网络框架，其核心开发团队来自日本的一家机器学习创业公司Preferred Networks。

PaddlePaddle

PaddlePaddle是百度开发的。

补充

和Tensorflow、Theano、Caffe等框架使用的静态计算图相比，动态计算图可以在运行时动态地构建计算图，因此非常很适合进行一些复杂的决策或推理任务。

Chainer、PyTorch使用动态计算图。

选PyTorch的原因

2019年5月推出了PyTorch 1.1版本。

  1. 目前比较火的深度学习框架是TensorFlow和PyTorch（如果要用TensorFlow，推荐用TensorFlow 2，不是TensorFlow 1）。
  2. PyTorch更有利于研究人员、爱好者、小规模项目等快速搞出原型。而TensorFlow更适合大规模部署，特别是需要跨平台和嵌入式部署时。
  3. 易用性高。

  1. 采用Python语言、动态图机制、网络构建灵活、拥有强大的社群。

深度学习以神经网络为主要模型，但神经网络和深度学习并不等价

深度学习可以采用神经网络模型，也可以采用其它模型。

深度学习主要以神经网络模型为基础，原因是神经网络模型可以使用误差反向传播算法，从而可以比较好地解决贡献度分配问题。
